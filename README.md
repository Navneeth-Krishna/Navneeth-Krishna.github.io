# Navneeth Krishna Aravind — Portfolio

Personal portfolio website for GitHub Pages.

## Files

- `index.html` — portfolio content and structure
- `style.css` — responsive dark-theme styling
- `script.js` — mobile navigation
- `assets/resume.pdf` — add your resume here

## GitHub Pages

1. Create a repository named `<your-github-username>.github.io`.
2. Upload all files to the repository root.
3. Put your resume at `assets/resume.pdf`.
4. Go to Settings → Pages.
5. Select `main` branch and `/ (root)`.
6. Open the published URL.
